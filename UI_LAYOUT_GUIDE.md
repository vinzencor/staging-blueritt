## Color Scheme

| Element | Color | Hex |
|---------|-------|-----|
| Avatar Background | Purple to Pink Gradient | #9333ea → #ec4899 |
| Followers Badge | Blue | #3b82f6 |
| Posts Badge | Purple | #a855f7 |
| Verified Badge | Blue | #3b82f6 |
| Card Border | Gray | #e5e7eb |
| Card Hover Border | Purple | #c084fc |
| Text Primary | Dark Gray | #111827 |
| Text Secondary | Gray | #4b5563 |

| Breakpoint | Width | Widget Width | Position |
|-----------|-------|--------------|----------|
| 2XL | ≥1536px | 320px | Fixed right-6 |
| XL | ≥1280px | 288px | Fixed right-4 |
| LG | ≥1024px | 256px | Fixed right-3 |
| MD | ≥768px | Full | Bottom drawer |
| SM | <768px | Full | Bottom drawer |
